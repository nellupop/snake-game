# Snake Game

A simple, playable Snake game built with vanilla HTML5 Canvas and JavaScript — no dependencies, no build step.

## Play

Open `index.html` in any modern browser and hit **Start Game**.

Or play it live via GitHub Pages:
https://nellupop.github.io/snake-game/

## Controls

| Key | Action |
|-----|--------|
| Arrow Keys or WASD | Move the snake |
| P | Pause / Unpause |

## Features

- 20×20 grid with smooth canvas rendering
- Score tracker with persistent best score (localStorage)
- Speed increases as your score grows
- Game over screen with replay button
